One system. Every employer, every invite, year-round.
The primary principle is one system — a single enrollment model that runs identically for new ERs and renewals, in January and in November. Designing for two CSDs is a strategic way of building that system well: it embeds the extension concept structurally rather than operationally. But the CSD guarantee is a feature of the design, not the principle itself.
The system
30-day window · marketed
Every invite gets a 30-day enrollment window. This is the number we communicate to employers and members. Same rule, every invite, all year.
+5 day last-chance window
On day 31, a last-chance nudge goes out. Members have 5 more days to enroll. This applies to new ERs and renewals alike — the mechanism is identical, only the durations differ.
Renewals and new ERs are the same system
Soft close + 5-day hard close. The logic is identical. Renewals have fixed calendar dates (OE opens Nov 2 or Nov 9); new ERs have a rolling 30-day window from invite date. One model, two configurations.
What changes
✕14-day, 21-day, and 60-day employer window options
✕Employer-set enrollment deadlines as default
✕Manual mass extension operations
✕Separate OE operating mode
✓30-day marketed window · same for everyone
✓+5 day last-chance extension · structural, not manual
✓EOM−2 processing cutoff · single operational rhythm
✓Renewals: allowances set by Oct 25
Open question · not yet decided
Jan 4–8 special extension
For the Nov 9–25 OE cohort, the 35-day window doesn't reach the Dec 29 cutoff. A Jan 4–8 special extension would give these 17 dates a second CSD (Feb 1).
This would break the one-system principle — it's a named exception for a specific cohort. Worth considering, but the tradeoff is explicit: better member experience for 17 OE dates vs. system purity. Not a decision yet.

How it works
Each invite starts a 30-day window (soft close). On day 31 a last-chance nudge goes out — members have 5 more days to enroll. The processing cutoff (EOM−2, adjusted to Thu or earlier) splits the full 35-day window:

Enroll by soft close (day 30) → CSD 1 if before cutoff
Days 1–35 (after cutoff) → CSD 2
